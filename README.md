📧 Email Classification ML
📌 Overview

Machine learning project to classify emails into Spam, Updates, and Social using NLP techniques and structured features.

🚀 Features
Text preprocessing
TF-IDF feature extraction
Logistic Regression & Random Forest
Performance evaluation (accuracy, precision, recall, F1-score)
⚙️ Tech Stack

Python, Pandas, Scikit-learn, NLP

▶️ Usage

python main.py

📈 Result

Achieves strong accuracy, especially in detecting spam emails.
